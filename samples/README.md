In notes.md sollte pro Test kurz festgehalten werden:

- verwendeter Prompt,
- Bildgenerator und Modell,
- was stilistisch funktioniert,
- was aus der Weltlogik herausfällt,
- welche Variante als neue Referenz taugt.
